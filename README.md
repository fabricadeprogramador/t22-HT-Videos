# t22-ht-videos

## Ao baixar o projeto execute o comando abaixo para baixar as dependências.
```
npm install
```

### Iniciar Aplicação
```
npm run serve
```

### Preparar projeto para produção
```
npm run build
```

### Executar Testes
```
npm run test
```

### Executar eslint
```
npm run lint
```
